## Getting Started

### Prerequisites

- Node (v16.10.0+)

- Npm (v7.24.0+)

<br />

### How To Use

```
git clone https://github.com/itsemirhanengin/mental-health-wellness-mobile-app.git
```

```
cd mental-health-wellness-mobile-app
```

```
yarn install
```

This will start the metro bundler:
```
yarn start
```

This will start the simulator:
```
yarn ios
```
